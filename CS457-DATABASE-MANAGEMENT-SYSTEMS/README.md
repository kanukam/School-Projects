Kennedy Anukam's relational database management system that can take in sqlite3 statements as input.

Can handle:
  - Creating a database
  - Deleting a database
  - Creating a table
  - Deleting a table
  - Select statements
  - Table insertions
  - Updating a table
  - Deleting tuples from a table
  - Inner and Outer Joins

### Tech
  - Python3
