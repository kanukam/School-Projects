# Sqlite3 Emulator

Can handle:
  - Creating a database
  - Deleting a database
  - Creating a table
  - Deleting a table
  - Select statements
  - Table insertions
  - Updating a table
  - Deleting tuples from a table
  - Inner and Outer Joins

### Tech
  - Python3

### Run Instructions
  - standard input: python3 database.py < (test script) (enter)
    Or
  - Python 3 interpreter
  
Linux and Windows compatible.
